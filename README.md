# @genart-dev/cli

Command-line interface for [genart.dev](https://genart.dev) — render, validate, export, and scaffold generative art sketches.

Part of [genart.dev](https://genart.dev) — a generative art platform with an MCP server, desktop app, and IDE extensions.

## Install

```bash
npm install -g @genart-dev/cli
```

## Usage

```bash
genart <command> [options]
```
